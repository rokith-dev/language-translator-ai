# 🌍 Language Translator AI

An AI-powered multilingual translation application built using Python, Streamlit, and MySQL.

The application allows users to:

- Translate English text into multiple languages
- Upload PDF documents and translate their contents
- Download translated PDF files
- Convert translated text into speech
- Store translation history in MySQL

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/rokith-dev/language-translator-ai.git
```

Move into project directory:

```bash
cd language-translator-ai
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
